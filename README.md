## Build Requirements

-   [Meson](https://mesonbuild.com)
-   C build system, I suggest [Ninja](https://ninja-build.org/)
-   C compiler, I suggest [Clang](https://clang.llvm.org/)

## Building via Meson

```
meson setup build
cd build
ninja
```
